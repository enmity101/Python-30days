# 🐍 30 Days of Python

Welcome to my 30 Days of Python journey 🚀

In this repository, I will share what I learn in Python every day from Day 1 to Day 30.
Each day I will upload notes, examples, and practice code.

## 📌 About This Challenge

The goal of this challenge is to:

* Learn Python step by step
* Practice coding daily
* Build small projects
* Stay consistent for 30 days

## 📖 What I Will Cover

I will start from basic topics like introduction, variables, data types, and operators.
Then I will move to strings, lists, dictionaries, loops, and functions.
After that, I will learn advanced topics like OOP, file handling, modules, APIs, and libraries.
At the end, I will build small projects using what I learned.

## 🎯 Goal

By the end of 30 days, I want to feel confident writing Python programs and solving problems.

* Python 3
